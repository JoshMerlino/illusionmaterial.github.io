Initializing Material

Material.init(OBJECT  options)

Options

option accepts default description
scrollbars boolean true show Material scrollbars instead of default
context boolean false show Material right click menus
nowaves boolean false prevent Waves.js from in initializing
detached array ["[md='false']"] an array of query selectors to ignore in materializing
selectionColor color #c1c1c1 color of the users selected text background
